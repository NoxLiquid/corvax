# _Modifications

Этот каталог (и связанные пространства имён вида `Content.*._Modifications`) содержит авторские изменения/код Nox (KloopRE), если в конкретных файлах не указано иное.

Этот файл:
- отмечает авторство;
- не меняет и не отменяет лицензии/условия исходного проекта или апстрима.

Если вы хотите использовать эти изменения вне рамок лицензии репозитория (например, в коммерческих целях) — согласуйте это отдельно с автором.
